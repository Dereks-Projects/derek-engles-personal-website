Somm Tips is a wine recommendation application that helps everyone pick better wines. This project was created with React (JavaScript), with customized CSS elements and a wine characteristic database that powers the application.

Somm Site

Somm.Site is a comprehensive digital education platform designed for wine enthusiasts and hospitality professionals seeking to elevate their knowledge and service skills. The site was developed using Wix with custom CSS enhancements, supported by Canva-designed media and a multi-platform social media campaign.

visit somm.site

Distributor Post

Distributor Post is an online publication that delivers articles, insights, and educational content tailored to beverage sales professionals across the global distribution industry. It is a custom React application that integrates RSS feeds, JavaScript interactivity, and branded visuals to support a focused digital media experience.

visit distributorpost.com

Wine Quizzes

WineQuizzes.net is a curated collection of wine-related questions developed over two decades, used to support sommelier training and exam preparation. The platform is a standalone web application built using HTML, CSS, and vanilla JavaScript, offering multiple quiz levels and interactive functionality.

visit winequizzes.net
